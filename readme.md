# Kanban Mini (jQuery)

A lightweight Kanban / Trello-style board built with **jQuery** and **jQuery UI**.

## Features
- Add, rename, delete columns
- Add, edit, delete cards (title, description, tag)
- Drag & drop cards between columns and reorder
- Reorder columns by drag
- Persist board in `localStorage`
- Export / Import board as JSON
- Responsive, keyboard-friendly interactions

## Tech
- HTML, CSS
- JavaScript (jQuery 3.x + jQuery UI)

## How to run
1. Clone or download the folder.
2. Open `index.html` in a browser.

## Notes
- Data is stored in your browser's `localStorage` under `kanban_mini_v1`.
- Use Export / Import to move boards between machines.
